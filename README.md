# 🖌️ OpenCV Mini Project: Mekus Virtual Canvas

## 📋 Requirements
- Python `3.12` or earlier
- Install dependencies from `requirements.txt` using:
  ```bash
  pip install -r requirements.txt
## 🧠 Project Overview
Mekus Virtual Canvas is a beginner-friendly project designed to demonstrate foundational skills in:

- Image processing
- Filtering
- Drawing
- Object tracking
- Object detection
- Gesture-based control

Built with two core modules:

- OpenCV – for efficient image handling and processing
- MediaPipe – for real-time hand tracking and gesture detection

*This project offers a hands-on introduction to computer vision concepts using intuitive controls and visual feedback.*

## 🛠️ Getting Started
To run the project:
1. Ensure Python 3.12 or below is installed
2. Install required packages
3. Launch the main script to start the virtual canvas

## ✨ Key Features
- Draw and change filters using finger gestures tracked in real time
- Switch colors and modes via simple hand positions
